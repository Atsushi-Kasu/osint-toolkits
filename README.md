# OSINT Toolkit 🔍

Advanced Open Source Intelligence (OSINT) Framework with 25+ tools organized in 5 categories.

## 🎯 Features

- **5 Specialized Categories**
  - 🔎 Reconnaissance & Information Gathering
  - 📱 Social Media Intelligence
  - 🌐 Network & Domain Intelligence
  - 📧 Email & Phone Intelligence
  - 🕵️ Advanced & DarkWeb Intelligence

- **25+ Premium Tools** integrated in one framework
- **Beautiful CLI Interface** with color-coded menus
- **Easy Navigation** and tool execution
- **Automated Installation** of all dependencies
- **Cross-Platform** (Termux, Linux, WSL)

Made by Atsushi

## 🚀 Quick Start

### Installation

```bash
# Clone repository
git clone https://github.com/Atsushi-Kasu/osint-toolkits.git
cd osint-toolkit

# Run installer
bash scripts/install.sh

# Run toolkit
python3 osint.py

For Termux

# Update packages
pkg update && pkg upgrade -y

# Clone and install
git clone https://github.com/Atsushi-Kasu/osint-toolkits.git
cd osint-toolkit
bash scripts/install.sh
python3 osint.py

### 🛠️ Tools Included
Category 1: Reconnaissance
- theHarvester - Email/subdomain/IP harvesting
- Sublist3r - Subdomain enumeration
- Amass - Attack surface mapping
- Recon-ng - Reconnaissance framework
- SpiderFoot - Automated OSINT
Category 2: Social Media
- Sherlock - Username search (300+ platforms)
- Social Mapper - Facial recognition OSINT
- Twint - Twitter intelligence
- Instagram-OSINT - Instagram analysis
- OSRF - Multi-platform username search
Category 3: Network & Domain
- Nmap - Network scanning
- Shodan - IoT/server search
- DNSRecon - DNS enumeration
- Fierce - DNS reconnaissance
- WhatWeb - Web fingerprinting
Category 4: Email & Phone
- Holehe - Email to account finder
- PhoneInfoga - Phone number OSINT
- Infoga - Email information gathering
- EmailHarvester - Email scraping
- h8mail - Data breach search
Category 5: Advanced
- OnionSearch - Dark web search
- Photon - Web crawler
- Metagoofil - Metadata extraction
- Exiftool - Image metadata
- Maltego - Link analysis

### 📖 Usage

# Start the toolkit
python3 osint.py

# Or create alias
echo "alias osint='python3 $(pwd)/osint.py'" >> ~/.bashrc
source ~/.bashrc
osint
